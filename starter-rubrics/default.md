# 默认 Rubric（文章评分标准）

## 1. 新颖度 (Novelty)
- 0: 老生常谈
- 3: 有独特视角
- 5: 颠覆性观点

## 2. 深度 (Depth)
- 0: 表面介绍
- 3: 有实际案例
- 5: 体系化分析

## 3. 可操作性 (Actionability)
- 0: 纯概念
- 3: 有方法论
- 5: 可复制步骤

## 4. 时效性 (Timeliness)
- 0: 过时话题
- 3: 近期热点
- 5: 独家首发

## 5. 清晰度 (Clarity)
- 0: 晦涩难懂
- 3: 通俗易懂
- 5: 深入浅出

## 6. 互动性 (Engagement)
- 0: 枯燥无味
- 3: 有亮点
- 5: 引人入胜

## 7. 可信度 (Credibility)
- 0: 主观臆断
- 3: 有数据支撑
- 5: 一手资料

## 8. 相关性 (Relevance)
- 0: 目标受众不匹配
- 3: 基本匹配
- 5: 精确命中痛点

## 9. 文笔 (Craft)
- 0: 生硬
- 3: 流畅
- 5: 有文采
